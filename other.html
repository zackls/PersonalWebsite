NON-FUNCTION GRAPHER

This program takes in two expressions involving X and Y and outputs all the points on a graph where they are equal. The original Flash version of this program can be found HERE, and the Java version can be found HERE.

A regular graphing calculator graphs functions by plugging in increasing X values to the expression you provide it and finding their corresponding Y values, then connecting the lines between that (X, Y) value and the previous one. My program works by literally plugging in EVERY point on the graph and seeing if the two expressions of X and Y provided by the user are within a tolerance of each other (not strictly equal), and if they are, it graphs a black pixel. I haven't adapted the function to take in user input yet (the way you change the expressions being evaluated is by literally going into the code and changing them, which is more efficient anyways) but it still works perfectly fine, and it makes some awesome pictures. Some of my favorites are shown to the right, click on the picture if you'd like to cycle through them. Some differences between the Flash and Java programs are evident, including some artifacts which appear differently on each program's output as well as the generic background color of the image produced. The practical applications of this program are really minimal, it's pretty much just a cool picture machine. The axes of the graph aren't shown because I figured it somewhat ruined the raw picture.
